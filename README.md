Данное решение предназначено для автоматизации прохождения CAPTCHA Arkose Labs (Funcaptcha) с использованием Python и Playwright. В системе реализованы слои REST API, интеграция с внешними сервисами 2Captcha или AntiCaptcha, а также хранилище данных на базе SQLAlchemy.

Проект ориентирован на продакшн-использование — включает продвинутую архитектуру, разделение ответственности, обработку ошибок, логирование и масштабируемость.

---

## Структура проекта

- `api/` — REST API для взаимодействия с внешними системами и клиентами
- `captcha_solver/` — модули для решения CAPTCHA, интеграция с 2Captcha/AntiCaptcha
- `database/` — слой работы с базой данных (используется SQLAlchemy и asyncpg)
- `services/` — бизнес-логика, обработка авторизации и CAPTCHA
- `tests/` — модуль тестов, покрытие API и CAPTCHA
- `config/` — конфигурационные файлы, параметры окружения
- `requirements.txt` — список зависимостей (готов к использованию на продакшене)
- `Dockerfile` — контейнеризация решения

---

## Особенности реализации

- Использование асинхронных библиотек для высокой производительности (FastAPI, asyncpg)
- Интеграция с внешними CAPTCHA сервисами (2Captcha/AntiCaptcha) с учетом таймаутов и ошибок
- Гибкое конфигурирование через YAML и переменные окружения
- Логирование с использованием Loguru для отладки и мониторинга
- Поддержка масштабирования с помощью очередей задач и брокеров (например, Redis + Celery, если потребуется)
- Обеспечение безопасности — хранение ключей API и секретов в защищенных настройках
- Тестирование и CI/CD (скрипты в `tests/` и настройках)
